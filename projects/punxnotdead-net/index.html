PunxNotDead.Net

This project is a tribute to PunxNotDead.Net, a (mostly) monthly e-zine dedicated to punk music and culture that I ran as a high schooler from the late 1990s to 2001. The website serves as a digital homage, celebrating the vibrant community it fostered, sharing music reviews, and promoting the unyielding spirit of punk.

Key Features:

  • A reflective narrative on the creation and evolution of the original e-zine.
  • Engaging visuals and an accessible layout that captures the essence of punk culture.
  • Built with HTML and CSS, showcasing a simple yet effective design.

Though the original site no longer exists, this page honors its legacy and the connections it forged among like-minded individuals passionate about punk music.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • GitHub Pages
